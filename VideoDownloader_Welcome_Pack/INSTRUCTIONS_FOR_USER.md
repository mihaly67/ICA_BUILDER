## Utasítás a Felhasználónak

Mivel én (ebben a sandboxban) a Builder repohoz (`ICA_BUILDER`) vagyok kötve, közvetlenül nem tudom módosítani egy másik (a videodownloader) GitHub repo fájljait.

Azonban előkészítettem a `JULES_ICA_MANIFESTO.md` fájlt a `VideoDownloader_Welcome_Pack` mappába. Ezt a fájlt kell a másik repóba bemásolnod. Amikor a másik Jules ügynök elindul, fel kell szólítanod őt, hogy **olvassa el a repójában található `JULES_ICA_MANIFESTO.md` fájlt.** Ebből azonnal érteni fogja:
1. Melyik IP címen van a VPS.
2. Mi a RAG könyvtárának az útja (`/home/misi/video_downloader_RAG/`).
3. Milyen szabályokat kell betartania (Zero Trust).
